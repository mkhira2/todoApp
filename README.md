# The To-Do's

This full stack application allows users to log in and keep track of their done, undone,
and completed tasks. Enjoy!


## Screenshot
![To Do App](/dist/assets/images/todo.png)

## Installation

```
git clone https://github.com/mkhira2/todoApp.git
cd todoApp
npm install
mongod
npm run go

open localhost:3000 in the browser
```
